點我前往 Swagger UI

本專案使用 Spring Boot、Spring Data JPA 及 MySQL 進行開發。 由於我過去主要專注於後端開發，對前端技術較不熟悉，因此在短短兩天內自學 HTML 和 CSS，完成了這個說明頁面。 雖然尚未學習 JavaScript 和 Vue.js，但未來會持續補足相關技能。

在後端部分，我已使用 Spring Data JPA 完成了基本的 CRUD 功能。 由於貴公司要求使用 Stored Procedure，但 JPA 對其支援較為複雜，因此開發過程中考慮改用 Spring JDBC， 但發現若全面轉換，需重新實作大量方法，且可能影響開發時程。因此，目前我已在 MySQL 中編寫了一些 Stored Procedure， 作為後續優化的基礎。

另外，在 登入驗證 方面，我原本計劃使用 Spring Security 搭配 JWT， 雖然了解其運作原理（如 AOP、Filter、JWT），但由於尚未實際使用過 Spring Security， 需要額外時間學習與測試，因此此功能將作為後續優化的一部分，未來會補充並上傳至 GitHub。

這次的開發過程讓我對 Spring Boot、JPA、Stored Procedure 及前端技術有了更深入的理解，也更確信自己對後端開發的熱忱。 我非常認同貴公司重視人才培養的理念，並希望有機會加入貴公司，在實戰中持續學習與成長。
